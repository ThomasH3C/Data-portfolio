# Projet de nettoyage de données - Data Cleaning

Ce projet vise à nettoyer un dataset industriel en traitant les valeurs manquantes, les aberrations et en transformant les données pour les rendre prêtes à l'analyse.

## Étapes du projet :
- Inspection des données
- Traitement des valeurs manquantes
- Détection des valeurs aberrantes
- Normalisation des données

## Objectif :
Le but est de montrer comment transformer un dataset brut en un format propre et structuré pour l'analyse.
