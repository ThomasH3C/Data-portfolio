# Data Portfolio

## Introduction

Bienvenue dans mon portfolio d'analyse de données. Ce portfolio contient des projets que j'ai réalisés pour mettre en valeur mes compétences en traitement de données, analyse exploratoire, et visualisation.

## Compétences
- Python (Pandas, Numpy, Matplotlib)
- Power BI
- SQL
- Git/GitHub

## Projets

1. **Data Cleaning** - Nettoyage et préparation de données industrielles.
2. **Exploratory Data Analysis** - Analyse exploratoire de la performance des processus industriels.
3. **Business Dashboards** - Création de tableaux de bord Power BI pour le suivi des KPI opérationnels.
4. **Advanced Data Analysis** - Analyse avancée des goulots de production et simulation des améliorations.
